# temp/plans

该目录用于存放临时性方案、路线、建议和下一步执行计划类报告。

约定：

```text
1. 后续所有阶段性 review 结论、下一步方案、路线建议、临时执行计划默认放在本目录。
2. 本目录内容属于临时工作资料，不作为正式标准入口。
3. 如果某份方案成熟并需要长期维护，应再迁移到 docs/、ai-sdlc/、skills/ 或 ROADMAP.md 等正式位置。
4. 文件名建议包含主题和日期，例如 route-artifact-next-plan-20260702.md。
```
